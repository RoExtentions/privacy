Privacy Policy

Roblox Profile: Current Game • Last updated: 26 October 2025

This Privacy Policy explains how the Roblox Profile: Current Game browser extension (“the Extension”) handles information. We designed the Extension to respect your privacy and avoid collecting personal data.

Summary

  • We do not collect, store, or sell user data.
  • The Extension reads information on Roblox profile pages and requests game presence data from Roblox’s official APIs in your browser.
  • No personal information is transmitted to our servers (we don’t operate any).

Information We Collect

None. The Extension does not collect, store, or transmit personally identifiable information, financial data, authentication data, health data, precise location, web history, or user activity for tracking.

How the Extension Works

When you visit a Roblox user’s profile, the Extension may call Roblox’s official APIs (e.g., presence and game details) using your existing Roblox session to determine if the user is in a joinable game. All processing occurs locally in your browser. The Extension may briefly cache the current game page URL in-memory to improve performance; this is not persisted or transmitted to any third party.

Permissions

  • contextMenus: Adds a right-click option “Open game page”.
  • activeTab: Interacts with the currently viewed Roblox profile tab to show status.
  • storage: May store minimal, non-personal settings or temporary values to reduce repeated API calls.
  • host_permissions: Accesses Roblox domains solely to retrieve presence and game details necessary for the Extension’s single purpose.

Data Sharing & Selling

We do not sell or transfer user data to third parties. We do not use data for advertising, profiling, or unrelated purposes.

Data Retention

The Extension does not retain personal data. Any temporary values used for functionality are kept locally and discarded when no longer needed.

Children’s Privacy

The Extension does not collect personal information from anyone, including children under 13.

Security

Because we do not collect or transmit personal data, risks are minimized. Requests to Roblox APIs occur over HTTPS using your existing session.

Your Rights

Since we do not collect personal data, there is nothing to access, correct, delete, or export. If you have questions, contact us using the details below.

Changes to This Policy

We may update this Policy to reflect improvements or changes in browser/Roblox platforms. Material changes will be reflected by updating the “Last updated” date above.
